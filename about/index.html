# 自我介绍

  • 关于一名改行自学（高龄 / 初级）前端开发者的自我叙述：

    • 记得第一次正式接触web前端行业是在2020年年初，2019年改行做前端入门之前报了培训班，怀揣着好奇的心思开始正式我的前端道路，为什么培训班被喷的这么惨，都是自己作的，招生根本不管那些适不适合做，他们都是在想着自己 能拿多少少是多少，搞个简单的入学考试（是个人都能过的）能让你感觉到就是筛选后才有的资格学习的，但是有一点无可否认就是培训班里面的学习气氛比你自己自学是有好处的；

    • 很多人感觉自己技术方向比较好，不与人交流，生怕别人学会了，其实这样想是不对的。技术只有在交流中才能得到提升，表述自己有利于理解的，可以利用小黄鸭学习法。利于发现自己的不足。自己埋头学习是不好的。如果每当你在学习后/解决了一个bug后通过交流/ 博客分享给别人，那么你的技术理解能力会更加的好。

    • 然而最大的难题就是：光看不写 / 敲。对于改行也好 、 初级前端也罢：当你在学习/工作中从头看到尾都能看懂,真正需要肌肉的练习和动手敲，不断地重复记忆法，才能流畅的写出代码编程。做前端真正厉害的不是你代码敲得多快，代码写的多少，而是写项目的逻辑思维和解决问题的能力而这种能力只能通过不断的深入思考和实际代码才能慢慢积累出来的。

  • 自己也很珍惜每一次的在职机会，做大的付出自己的能力范围之内的技术。
